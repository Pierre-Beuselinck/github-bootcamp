## 👨‍🚀 Exercise 1

- [ ] Create a Push Event

```yml
name: CI on Push
on:
  push:
    branches:
      - main

jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - name: Checkout code
        uses: actions/checkout@v2
      - name: Run a one-line script
        run: echo Hello, Push Event!
```

- [ ] Commit to your repository to launch the action

## 👨‍🚀 Exercise 2

- [ ] Create a Pull Request Event

name: CI on Pull Request

  pull_request:

      - master

    runs-on: ubuntu-latest
    steps:
