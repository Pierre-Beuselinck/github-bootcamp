## How to initialise a new repository

```
mkdir mon_projet
cd mon_projet
git init

git add --all
git commit --message
Git push
```

---

## How to review changes

Si on constate des modifications avec 'git status', il est possible de les inspecter avec 'git diff' : 

```
git diff
git diff <file_name>
```
